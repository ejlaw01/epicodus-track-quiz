# Epicodus Tracker #### _What track should I follow?_ ### By _**Ethan Law**_ ### Description _This website includes a quiz to help users determine what track they should follow while enrolled at vocational programming school Epicodus. Upon completion of the quiz, information is given about the courses said student would be taking in a given track._ ### Setup * _Clone from Github_ * _Run on a web browser_ * _View_ _or just click [here](ejlaw01.github.io/epicodus-track-quiz) ### Technologies used _HTML, CSS, Bootstrap, JavaScript and jQuery_ ### Known bugs _Quiz favors certain tracks (e.g. CSS/Design) over others in the case of a tie._ ### License _Default_
